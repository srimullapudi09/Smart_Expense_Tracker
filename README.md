#💰 Smart Expense Tracker Dashboard

A responsive and interactive Expense Tracker Dashboard built using HTML, CSS, and JavaScript that helps users manage their daily expenses efficiently. The application allows users to add, edit, delete, search, and filter expenses while providing real-time analytics through charts and summary cards. All data is stored locally using the browser's Local Storage, ensuring persistence even after page refreshes.

#🚀 Features
Add, Edit, and Delete Expenses
Search Expenses by Title
Filter Expenses by Category
Real-Time Expense Analytics
Interactive Pie Chart Visualization
Total Expense, Transaction Count, and Highest Expense Tracking
Dark Mode with Theme Persistence
Export Expense Data to CSV
Responsive Design for Mobile and Desktop
Local Storage for Data Persistence
Toast Notifications for User Actions

#🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Chart.js
Local Storage API

#📊 Key Learnings

#🎯 Project Objective

The objective of this project is to provide a simple yet effective solution for personal expense management while demonstrating core front-end development concepts and JavaScript functionality.
