# 💰 Smart Expense Tracker Dashboard

## 📌 Overview
Smart Expense Tracker Dashboard is a responsive web application built using HTML, CSS, and JavaScript that helps users manage and analyze their daily expenses. The application provides an intuitive interface for tracking spending, categorizing expenses, visualizing financial data, and exporting records for future use.

---

## 🚀 Features

- ✅ Add, Edit, and Delete Expenses
- ✅ Search Expenses by Title
- ✅ Filter Expenses by Category
- ✅ Real-Time Expense Analytics
- ✅ Interactive Pie Chart using Chart.js
- ✅ Total Expense Tracking
- ✅ Highest Expense Detection
- ✅ Transaction Count Summary
- ✅ Dark Mode with Theme Persistence
- ✅ Export Expenses to CSV
- ✅ Local Storage Data Persistence
- ✅ Toast Notifications
- ✅ Fully Responsive Design

---

## 🛠️ Tech Stack

- JavaScript (ES6)
- Chart.js
- Local Storage API

## ⚙️ How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Open `index.html`.
5. Click **Open with Live Server**.
6. Start managing your expenses.
